# Industrial Cyber-Physical Systems Architect | OT Cybersecurity

Especialista em segurança de infraestruturas críticas e resiliência de sistemas industriais. Traduzo os requisitos regulatórios de alta governança (NIST SP 800-82r3, ISA/IEC 62443) em arquiteturas de rede blindadas e lógicas de controle imutáveis para mitigar riscos cinéticos no chão de fábrica.

## 🚀 Key Performance Indicators (KPIs)

* **Cyber-Physical Resilience:** Projeto de arquiteturas de redes industriais baseadas no Modelo Purdue e IDMZ, reduzindo a superfície de ataque lógico e mitigando riscos de movimentação lateral de malware (ex: Ransomware).
* **Financial Risk Mitigation:** Engenharia de controles compensatórios para ativos legados e obsoletos (*unpatchable OS*), evitando paradas de linha catastróficas com impacto financeiro estimado em mais de R$ 149.500,00/hora.
* **Integrity Enforcement:** Implementação de políticas rígidas de segurança de software e auditoria de dados (*Audit Trail*) no nível de Field I/O (Purdue Level 0) utilizando conceitos de Digital Twins para detecção de Spoofing de processos.
